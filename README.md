Employee Management System

A full-stack Employee Management System designed for admin users to efficiently manage employee data through a clean, responsive, and accessible interface.

✨ Key Features

Secure admin authentication using email and password

Employee management with full CRUD operations

Searchable and paginated data tables for better usability

Responsive, mobile-first UI with accessibility best practices

🛠️ Tech Stack

Frontend: React, Next.js, TypeScript, Tailwind CSS
Backend: Node.js, Express
Database: PostgreSQL

🚀 Getting Started
Prerequisites

Node.js (v18+ recommended)

PostgreSQL

Installation
# Clone the repository
git clone https://github.com/your-username/employee-management-system.git


# Install dependencies
npm install


# Start the development server
npm run dev
